Big Ideas Math (BIM) Instructions

Follow these steps to use the Nova tool on Big Ideas Math:

Overview

Nova provides real-time assistance on Big Ideas Math assignments.

Features

  • Automatic Question Analysis: Detects question type (multiple-choice, True/False, fill-in-the-blank) and suggests answers.
  • Clean UI: Unobtrusive overlays, draggable panels (via jQuery UI).

Installation & Usage (Bookmarklet)

  1. Drag the "Nova" bookmarklet (found in the section below) to your browser's bookmarks bar.
  2. Ensure you are on a BIM assignment page (bigideasmath.com).
  3. Click the "Nova" bookmarklet while on the assignment page.
  4. A floating UI will appear on the page.
  5. Click the "Get Answer" button on the floating UI to reveal the answer(s) for the current question.
  6. The answer will also be copied to your clipboard.

IXL Instructions

For IXL, we recommend using the dedicated browser extension for the best experience. Follow the steps below to install and use the extension in your browser.

Overview

The Nova IXL browser extension enhances your IXL experience with real-time assistance.

Features

  • Progress Analytics: Tracks performance and offers hints in real time.
  • Clean UI: Unobtrusive overlays.

Installation & Usage (Extension)

For Chrome:
  1. Download the latest release from GitHub Releases.
  2. Unzip the downloaded file to a folder (e.g., ixl extension).
  3. Open Chrome Extensions (chrome://extensions/). Enable Developer mode (top right).
  4. Click Load unpacked and select the ixl extension folder.
For Firefox:
  1. Visit the Mozilla Add-ons page.
  2. Click "Add to Firefox" and follow the installation prompts.
  3. Once installed, look for the Nova IXL icon in your toolbar.
Need help? See the GitHub repository for more details.

DeltaMath Instructions

Follow these steps to use the Nova tool on DeltaMath:

Overview

Nova provides assistance on DeltaMath problems by extracting answers.

Features

  • Floating UI panel showing KaTeX-rendered answers.
  • Dynamically injects necessary libraries (KaTeX, jQuery, jQuery UI) if missing.

Installation & Usage (Bookmarklet)

  1. Drag the "Nova" bookmarklet (found in the section below) to your browser's bookmarks bar.
  2. Ensure you are on a specific DeltaMath problem page (where you see the problem and input fields) (deltamath.com).
  3. Click the "Nova" bookmarklet while on the problem page.
  4. A floating UI will appear on the page, displaying the extracted answer(s).
  5. The answer will also be copied to your clipboard.

Note: This tool extracts the correct answer directly from the page's underlying data structures when available. It may not work on all problem types or if DeltaMath changes its internal structure.
