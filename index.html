BIM Learning Assistant

Enhance your understanding and efficiency with the BIM Learning Assistant for Big Ideas Math assignments.

Key Features

  • Intelligent question analysis to aid comprehension.
  • Support for various question formats (multiple choice, T/F, fill-in-the-blank).
  • Integrates seamlessly with the Big Ideas Math platform.
  • Designed for a clean and non-intrusive experience.

Usage Guide

  1. Navigate to any Big Ideas Math assignment page.
  2. Click the "Nova" bookmarklet you saved to your bookmarks bar.
  3. The assistant will activate, providing support to enhance your learning process.

DeltaMath UI Helper

Streamline your DeltaMath experience with the UI Helper, providing a convenient floating panel for extracted information.

Key Features

  • Creates a floating, draggable UI panel.
  • Displays extracted data (e.g., answers) clearly using KaTeX.
  • Dynamically loads necessary libraries (KaTeX, jQuery, jQuery UI).
  • Enhances usability with smooth dragging animations.

Usage Guide

  1. Navigate to a supported DeltaMath assignment page.
  2. Click the "Nova" bookmarklet in your bookmarks bar.
  3. A floating panel containing extracted information will appear, which you can drag to reposition.

Documentation & Overview

Nova is a suite of browser tools designed to support and enhance your online learning experience across various platforms. Our goal is to provide intelligent assistance that helps you understand concepts and navigate assignments more efficiently.

Tools in the Suite

BIM Learning Assistant

  • Automatic question analysis for support.
  • Supports diverse question types.
  • Clean, integrated interface.

Achieve Learning Assistant

  • Intelligent assistance for Achieve.
  • Seamless platform integration.
  • Features for enhanced comprehension.

IXL Learning Companion (Coming Soon)

  • Real-time feedback & hints planned.
  • Progress tracking planned.
  • Seamless integration planned.

DeltaMath UI Helper

  • Floating UI for data extraction.
  • KaTeX rendering for math.
  • Draggable panel for convenience.

Core Features

  • Context-aware assistance for supported platforms.
  • Designed to support various question types.
  • User-friendly and non-obtrusive design.
  • Cross-browser compatibility where supported.

Supported Platforms

  • Big Ideas Math: Automatically loads the script on bigideasmath.com.
  • Achieve3000: Automatically loads the script on achieve3000.com.
  • DeltaMath: Loads the script on supported assignment pages on deltamath.com.
  • IXL: Support planned via browser extension (Coming Soon).

Troubleshooting

If you encounter issues:

  • Ensure your browser's bookmarks bar is visible and that you dragged the bookmarklet correctly.
  • Verify that you are currently visiting a supported site (check the URL).
  • Reload the page and try clicking the bookmarklet again.
  • Check your browser's developer console (usually F12) for error messages.
  • Consult the GitHub Issues page for known problems.
